<b>Instructor:</b> Dr. Marten van Schijndel

This class teaches some of the foundational techniques and theory behind computational language processing.

<b>Key topics:</b> 

<ul>
   <li> Finite State Machines </li>
   <li> Tokenization / Smoothing </li>
   <li> Probability Theory / Bayesian Methods </li>
   <li> Information Theory </li>
   <li> Hidden Markov Models </li>
   <li> Context-Free Grammars / Parsing Algorithms </li>
   <li> Semantic Roles / Semantic Frame Induction </li>
</ul>
